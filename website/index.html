Welcome to SimpleALARM

Updated on June 5th 2015 by Gerry Armstrong

SimpleALARM is a free web based alarm monitoring system for your Mitel 3300 PBXes. Yes, it is FREE!

Demo is now available, just follow the Login link and use demo/demo to login.

How does it work?

SimpleALARM uses a single executable that runs on a Windows PC or server that can reach the PBX(es) that are to be monitored via IP as well as connect to the internet. This executable uses SNMP to check the alarm state of the PBX and if there is a change of this alarm state it will update a database and send an alert via email to an address you provide with the new alarm state. You can also monitor the state of the PBX via any web browser.

How do you use it?

First you need to register with the service and then login to download the application. Once you have registered the link to the application and further instruction are available for download. Please note it is VERY IMPORTANT to match the company name you register with with the company name you enter in the config.ini file or you will not be able to view your PBX!

Doesn't Nagios already do this?

Sure it does, if you want to go through the trouble of installing a Nagios server on site, configure it for the PBX with alerting etc then Nagios will do the same thing. Then again, with this solution all you need to do is register, download a zip file, extract to a Windwos PC/Server, configure an ini file and run the app. After a few minutes you have a complete working alerting and visual display of the state of your PBXes. You can use Windows Task Scheduler schedule the command to check the state of the PBX every 5 mins or whatever interval you wish to set it to. Another advantage over application like Nagios is that you can monitor other PBXes outside of your network so if you are responsible for monitoring external PBXes then all you need to do is deploy the same download on a PC on the remote site.
